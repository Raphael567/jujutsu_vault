# 🌀 Jujutsu Vault

O **Jujutsu Vault** é um projeto web inspirado no universo de *Jujutsu Kaisen*, desenvolvido com o objetivo de reunir informações, interações e experiências relacionadas à obra em um só lugar.

A proposta do projeto é oferecer uma plataforma onde usuários possam explorar personagens, técnicas, curiosidades e até interagir por meio de quizzes e outras funcionalidades.

---

## 🚀 Propósito

O projeto foi criado com os seguintes objetivos:

* 📖 Centralizar informações sobre o universo de Jujutsu Kaisen
* 🎮 Proporcionar uma experiência interativa (como quizzes)
* 💻 Praticar e demonstrar habilidades em desenvolvimento web
* 🧩 Servir como base para evolução com novas features no futuro

---

## 📦 Como rodar o projeto

Siga os passos abaixo para executar o projeto na sua máquina:

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/jujutsu-vault.git
```

### 2. Acesse a pasta do projeto

```bash
cd jujutsu-vault
```

### 3. Instale as dependências

```bash
npm install
```

ou simplesmente:

```bash
npm i
```

### 4. Execute o projeto

```bash
npm start
```

ou, dependendo da configuração:

```bash
npm run dev
```
